## Low power CPU efficient Instance Segmentation model using Detectron2 framework

### Installation
```
* pip install pyyaml==5.1
* pip install 'git+https://github.com/facebookresearch/detectron2.git'
```
### Usage 
```
python test.py Carspotters.jpg
```
